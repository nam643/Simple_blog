# Simple Blog Web App

A simple full-stack blog web application built with **Node.js**, **Express.js**, and **EJS**, allowing users to create, view, edit, and delete blog posts using server-side rendering.

## Features
- Create new blog posts via a form-based interface
- View all blog posts on a dedicated blog page
- Edit existing posts
- Delete posts
- Server-side rendering with EJS templates
- Clean routing and form handling with Express

## Tech Stack
- **Backend:** Node.js, Express.js
- **Frontend:** EJS, CSS, JavaScript
- **Middleware:** body-parser
- **Tooling:** Git, Nodemon

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm
